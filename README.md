# link-safety-analyzer-flutter
🔗 Link Safety Analyzer – Mobile App

Bu proje, Flutter ve Dart kullanılarak geliştirilmiş bir mobil güvenlik uygulamasıdır.

---

📌 Proje Hakkında

Kullanıcılar şüpheli olabileceğini düşündükleri linkleri uygulamaya yapıştırarak analiz eder ve linkin güvenli olup olmadığı hakkında bilgi alır. Amaç, kullanıcıların zararlı linklere tıklamasını önlemektir.

---

⚙️ Kullanılan Teknolojiler

- Flutter
- Dart
- API entegrasyonu (varsa belirt)
- Mobile UI/UX tasarımı

---

🚀 Özellikler

- Link kopyala / yapıştır ile analiz
- Şüpheli link kontrol sistemi
- Kullanıcı dostu mobil arayüz
- Hızlı sonuç gösterimi

---

🎯 Amaç

Kullanıcıları zararlı, phishing veya şüpheli linklere karşı bilinçlendirmek ve mobil ortamda temel bir güvenlik kontrol sistemi sunmak.

---

🧠 Geliştirme Aşaması

- Algoritma ve doğruluk geliştirme aşamasındadır
- Daha gelişmiş güvenlik analiz sistemleri üzerinde çalışılmaktadır
